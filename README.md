# Notion Widget Pomodoro Timer

A Pomodoro timer designed for seamless integration into Notion. This widget allows you to track your work and break sessions with customizable session lengths and break lengths. The timer is mobile-friendly, supports both light and dark modes, and is optimized for use within Notion.

## Features

- **Responsive Design**: Optimized for both desktop and mobile views, adapting perfectly to various screen sizes.
- **Light & Dark Mode**: Automatically adjusts to match the user's system theme (light or dark mode).
- **Customizable Timings**: Allows you to set custom session lengths and break lengths.
- **Simple and Intuitive**: The user interface is clean and easy to interact with, making it a great productivity tool.

## Installation

### Prerequisites

- A modern web browser that supports HTML5, CSS3, and JavaScript.
- A Notion account for embedding the widget.

### Steps to use in Notion:

1. Clone or download the repository.
2. Open the `index.html` in your browser to test the Pomodoro timer.
3. To use it in Notion:
   - Embed the widget by adding the iframe or using an embed block for external content. Simply copy the widget's URL or use a third-party service to host and embed it.

## Usage

### Configuring the Timer
- You can change the **session length** and **break length** by clicking the up/down arrows next to each field.
- Start the timer by clicking the **Play** button, and pause it using the **Pause** button.

### Dark Mode Support
- The timer dynamically adjusts to dark mode when the system theme is set to dark. It ensures that the widget is visible and usable in all environments.
